# AI Agent工程内的降级策略

在 AI Agent（智能体）工程里，**“降级”**说白了就是一句大实话：**“惹不起，我还躲不起吗？”**

它的准确定义是：**当 Agent 的核心能力（比如最强模型、复杂工具、向量检索）因为超时、报错或成本过高而走不通时，系统自动切换到一个“更笨、更慢、更便宜但绝对能跑通”的备用方案，保证业务不中断。**

这和普通互联网后端（如支付降级）有本质区别——**普通后端降级是“牺牲体验保存活”，Agent 降级是“牺牲智商保交卷”。**

这里将Agent 里最常见的 5 种“降级姿势”彻底拆开看：

---

### 1. 模型降级（最常见）：从“博士生”换到“流水线工人”
-   **正常情况**：调用 GPT-4o 或 Claude 3.5 Sonnet（贵、慢、强）处理所有复杂推理。
-   **触发异常**：API 超时、频率限制（Rate Limit）、或单次消耗 Token 过多导致费用爆炸。
-   **降级动作**：**立即切换到 GPT-3.5-Turbo 或 DeepSeek-V3 等轻量模型。**
-   **代价**：推理能力变差，可能写不出复杂算法，但能勉强给出“伪代码”或“思路建议”，**用户至少不面对白屏报错**。

### 2. 上下文降级（呼应你的“128K 窗口”）：从“全量记忆”切到“摘要便签”
-   **正常情况**：把 100K 的历史对话 + 代码全文一股脑扔进窗口（追求高精度）。
-   **触发异常**：当你准备让 AI 生成长文档时，发现 `历史 + 提问 + 预估输出` > 128K 上限（就像你上一轮问的场景）。
-   **降级动作**：**放弃“全文窗口”策略**，切换到你之前理解的**长期记忆检索（RAG）**——只把最相关的 5 段代码片段（5K）塞进去，并且强制把历史压缩成 1K 的摘要。
-   **代价**：AI 可能丢失部分细节变量名，但**保证了能正常生成输出，不会直接报错截断**。

### 3. 工具调用降级（Tool Call Fallback）：从“精密手术刀”换成“大锤”
-   **正常情况**：Agent 调用 `get_weather(经纬度, 时间)` 精确 API。
-   **触发异常**：该 API 挂了，或返回数据格式解析失败（JSON 乱码）。
-   **降级动作**：**放弃调用该工具**，转而用搜索引擎爬取“地区+天气”的纯文本网页，或用内置的正则表达式粗暴截取上一轮缓存里的旧数据。
-   **代价**：数据精度降低（可能是昨天的天气），但**用户能拿到一个大概数值，而不是“工具调用失败”的红字**。

### 4. 检索降级（向量变全文）：这就是你问过的 OpenClaw 场景
-   **正常情况**：使用 **LanceDB** 进行语义向量检索（最准，但吃 GPU 显存）。
-   **触发异常**：LanceDB 索引损坏、OOM（内存溢出）或响应超时（超过 5 秒）。
-   **降级动作**：**立即降级回 Hermes 用的 SQLite+FTS5（全文关键词检索）**。这也是 OpenClaw 设计里“QMD 不可用时降级到内置 SQLite”的底层逻辑。
-   **代价**：搜不出来“把用户登录态刷新一下”这种意图，但**一定能搜出代码里带 `refresh_token` 关键字的文件**，找回率下降，但**绝不挂掉**。

### 5. 解析降级（Parse Fallback）：从“结构化”变“瞎蒙”
-   **正常情况**：要求 Agent 必须返回合法的 JSON 格式，方便前端渲染图表。
-   **触发异常**：模型抽风，返回了一堆 Markdown 夹杂着 `{"a": 1` 这种不完整的字符串。
-   **降级动作**：**放弃 JSON 解析器**，改为直接用正则表达式 `/\{.*\}/` 暴力截取大括号里的内容，甚至直接把模型输出的纯文本展示在界面上。
-   **代价**：前端没法做漂亮的控件，但**用户至少能看到 AI 算出来的数字结果**。

---

### 💎 对设计产品（或做平台）最重要的工程心法

在做 Agent 架构时，**降级策略必须写在“系统运行手册（Runbook）”的第一页**，通常遵循 **“熔断器（Circuit Breaker）”** 模式：

1.  **自动探测**：连续失败 3 次，自动打开降级开关。
2.  **静默切换**：切换过程对用户透明（前端不报错，只可能感觉“回答变慢”或“变蠢”）。
3.  **后台重试**：降级之后，后台依然偷偷尝试恢复主链路（比如重连 LanceDB），一旦主链路恢复，下一个请求自动切回来。

**总结**：

> **降级就是给 Agent 准备的“B 计划”。上限是让 AI 拿 100 分，降级是哪怕 AI 头晕眼花、手边只有草稿纸，也得硬着头皮给你凑出 60 分及格，绝对不许交白卷。** 没有降级策略的 Agent，在线上跑就是一颗定时炸弹。

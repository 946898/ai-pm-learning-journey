## 一、整体阶段划分（当前仅展示前两阶段）

整个计划分为两大阶段，循序渐进：

1. **阶段一：基础搭建（W1–W4）**  
   核心目标是“能搭出一个可用的 Agent 原型”，掌握 LangGraph、Function Calling、MCP 协议和多 Agent 协同，并产出第一个可运行的采购审核 Agent + 物流查询 MCP 工具。

2. **阶段二：行业落地 + 作品集（W5–W8）**  
   核心目标是“把技术能力和行业 Know‑how 绑定”，通过深度拆解 3 个行业头部 Agent 案例、完成 1 个高质量的深做 Demo 并撰写 1 篇深度分析文章，形成差异化竞争力。

> **后续阶段**（求职冲刺）将在前两阶段完成后根据实际情况另行规划，暂不列出。
